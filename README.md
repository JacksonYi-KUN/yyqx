一、项目介绍（游戏核心玩法）
本次制作的 3D 小游戏是一款 “治愈系捡取类游戏”，核心玩法非常直观：
玩家通过 WASD 键盘按键控制场景中的小男孩前后左右移动；
小男孩靠近或碰到彩色方块时，会自动捡取方块（方块触发消失动画 + 得分增加 + 音效反馈）；
按 R 键可重置游戏（得分清零 + 重新生成所有方块）；
当所有方块被捡取完毕后，系统会自动刷新一批新方块，无需手动操作；
额外设计了治愈系视觉效果（渐变背景、草地、方块弹跳动画、小男孩静止微动），提升游戏体验。
游戏整体难度适中，操作逻辑清晰，既覆盖了 Three.js 的核心知识点，又有完整的交互闭环，符合 3D 入门小游戏的作业要求。
二、技术栈说明
本次项目未使用任何框架，仅基于原生技术开发，确保代码简洁、可追溯，具体技术栈如下：
核心 3D 引擎：Three.js（v0.x 版本，本地node_modules依赖，无 CDN 引入，避免路径报错问题）；
前端基础：HTML5（结构搭建）、CSS3（UI 样式 + 渐变背景）、原生 JavaScript（交互逻辑 + 动画控制）；
辅助技术：WebGL（Three.js 底层渲染）、Web Audio API（内置音效，无外部音频文件）。
所有资源均为本地或内置，无需联网加载外部资源，保证游戏运行稳定性。
